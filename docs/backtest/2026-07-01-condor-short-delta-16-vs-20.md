# 回测：铁鹰短腿 delta 16Δ vs 20Δ（含佣金净收益）

日期：2026-07-01 · 引擎 `run_condor_bs_backtest`（BS 重定价，B 方案）
· 同源数据/局限见 `docs/backtest/2026-06-29-condor-bs-pnl-results.md`（**平 IV 无 skew，低估尾部**，仅相对比较）。

> 目的：验证"把短腿从 16Δ 调到 20Δ 加厚权利金"是否更划算。**结论：不划算——20Δ 全面更差，且佣金吃掉毛利一半以上。**
> 起因：静态单笔算术曾提示"20Δ 权利金 +26%、净利近翻倍"，但那假设吃到固定 27% 权利金；回测推翻之。

## 参数（单一变量 = short-delta）
SPY、VIX−4 代理、2019-01-01→2026-05-01、target-dte 40、**dte-exit 15**、wing 5、
绝对闸 IV≥0.20、固定止盈、单仓顺序、1 张、multiplier 100。佣金：**每笔往返 $25.76**（4腿×2×$3.22，实盘实测）。复现：
```bash
for D in 0.16 0.20; do python3 -m option_bot.backtest --condor-bs --symbol SPY \
  --vix-csv data/VIX_History.csv --gap 4 --from 2019-01-01 --to 2026-05-01 \
  --target-dte 40 --dte-exit 15 --short-delta $D --wing 5 \
  --gate-mode absolute --min-iv 0.20 --close-strategy threshold; done
# 净 = 毛总 − 入场数 × 25.76（回测引擎输出为毛值，佣金手工扣）
```

## 结果（区间 7.3 年，含扣佣金）
| 指标 | 16Δ（现状）| 20Δ | 谁好 |
|---|---|---|---|
| 入场数 | 48 | 43 | — |
| 胜率 | **83.3%** | 76.7% | 16Δ |
| 毛总盈亏 | **$2157** | $1695 | 16Δ |
| 毛均值/笔 | **$44.9（31.1%权利金）** | $39.4（21.5%权利金）| 16Δ |
| 最大回撤 | **−$312** | **−$534** | 16Δ（20Δ 差近一倍）|
| profit_factor | **3.39** | 2.21 | 16Δ |
| 往返佣金合计 | $1236（=48×25.76）| $1108（=43×25.76）| — |
| **扣佣净总** | **$920** | $587 | **16Δ** |
| **净均值/笔** | **$19.2** | $13.7 | **16Δ** |
| **净/年（单张）** | **~$126** | ~$80 | **16Δ** |
| 佣金占毛利 | **57%** | **65%** | — |

## 结论
1. **20Δ 全面更差**：胜率、毛/净总、毛/净均值、最大回撤（近翻倍 −$312→−$534）、profit_factor，无一胜出。
2. **"厚权利金是幻觉"**：20Δ 每张收更多（$183 vs $146 mid），但**实际只吃到 21.5% 权利金**（16Δ 吃 31.1%）——短腿更近→**击穿更频繁**→止盈不干净、尾部更烂，把多收的权利金和更多都赔回去。
3. **静态单笔账误导**：它假设"吃到固定 27%"，但真实捕获率随 delta 升而降。**必须回测看全样本期望，不能信静态表。**
4. **佣金吃掉毛利一半以上**（16Δ 57%、20Δ 65%）：净收益单张 16Δ ~$126/年、20Δ ~$80/年——**且这是无 skew/滑点的乐观 BS，真实更低。**

## 决策
- **保持 `OBOT_CONDOR_SHORT_DELTA=0.16`**，不调 20Δ。
- **调 delta 解决不了佣金问题，还更糟**；佣金是结构性问题，只能靠加张数(换绝对金额，占比不变)/加翼宽(等比放大风险)/或接受低性价比。
- 加厚权利金与 IV 无关（delta 选腿），见策略总览 §7.8。

## 局限
- BS 模型、平 IV 无 skew → **低估尾部**：20Δ 更靠近现价、真实击穿/尾部只会比 −$534 更糟，即 20Δ 的劣势被**低估**。
- 佣金按实盘实测 $25.76/往返 手工扣；未计滑点/点差（真实再打折）。
